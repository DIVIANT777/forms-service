# Forms Service

Сервис с несколькими формами (блоками).  
Каждая форма сохраняет ответы в одну Google таблицу, но в разные листы.

## Как запустить

### 1) GitHub Pages
1. Создай репозиторий
2. Залей файлы
3. Settings → Pages → Source: main → root
4. Получишь ссылку на сайт

### 2) Google Apps Script
1. Открой Google Sheets
2. Extensions → Apps Script
3. Вставь код doPost
4. Deploy → Web App
5. Скопируй URL
6. Вставь в app.js в API_URL

Готово.
